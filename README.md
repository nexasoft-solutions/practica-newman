# práctica-newman

Este proyecto es parte de una práctica de la asignatura **Entornos de Ingeniería del Software** del Máster en Ingeniería de Software de la UNIR.

El objetivo es aplicar un flujo de trabajo con **Git y GitHub**, creando ramas, pull requests y colaborando sobre un repositorio compartido.

## 📌 Descripción del Script

El script `ordenar_palabras.py` permite ordenar alfabéticamente una lista de palabras introducidas por línea de comandos.

```bash
python ordenar_palabras.py gato perro abeja
